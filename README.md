
Git Network
===========

Just a sanbox app that load user repos on https://api.github.com/ and show it on a graph using Knockout + Gulp + Bootstrap + D3 + Less.

Install
-------

```
sudo npm i -g bower gulp
npm i
bower i
```

Test
----

```
npm test
```

Dev
---

```
gulp watch-less
```

Dist
----

```
gulp dist
```
